# 🎯 Project Overview

This project was developed for the Malaysia Data Innovation Talent × DOSM Datathon 2025 to address Malaysia's growing demand for skilled talent in the data centre industry.

The project combined educational, workforce, infrastructure, and economic datasets to identify the most suitable districts for deploying data centre micro-credential programmes. Using data analytics, machine learning concepts, geospatial analysis, and dashboard development, the project generated actionable recommendations for policymakers, TVET institutions, and government agencies.

---

# 🚀 Problem Statement

Malaysia is rapidly becoming a regional data centre hub, attracting investments from global hyperscalers such as AWS, Google, and Microsoft. However, the country faces a shortage of specialized talent in areas such as:

* Network Engineering
* Cybersecurity
* Data Centre Operations
* Energy Management
* Compliance & Governance

Without targeted workforce planning, Malaysia risks slowing its digital infrastructure growth and losing competitiveness in the global digital economy.

# 📈 Key Findings

The analysis identified the most promising districts for data centre talent development:

| Rank | District        | Recommended Skill Track            |
| ---- | --------------- | ---------------------------------- |
| 1    | Cyberjaya       | Network & Security                 |
| 2    | Iskandar Puteri | Facilities Operations & Energy     |
| 3    | Bayan Lepas     | Compliance & Cybersecurity         |
| 4    | Kulim           | Energy & Infrastructure Operations |
| 5    | Johor Bahru     | Multi-Track Deployment             |

These districts demonstrated strong combinations of STEM talent availability, infrastructure readiness, and proximity to existing data centre investments.
